À propos

Ce gabarit remplace le code exporté d'un constructeur de sites par une base HTML/CSS propre. Il n'embarque aucune dépendance externe, fonctionne hors ligne et s'adapte à GitHub Pages.

Astuce : gardez vos chemins relatifs (par ex. images/photo.jpg) pour que le site marche aussi bien en local que publiquement.

Fonctionnalités

Design réactif

Mise en page fluide, grilles CSS, composants simples.

Accessibilité

Navigation au clavier, contrastes, attributs alt et structure sémantique.

Images optimisées

Exemples <picture>, loading="lazy", dimensions fixées pour éviter le CLS.
